# 职场边界局 · 公众号爆款写作 Skill

一个用于 [Claude](https://claude.com/claude-code) 的自定义 Skill，帮助为微信公众号「职场边界局」创作文章。

主打 **职场情绪 / 个人成长 / AI 时代下打工人的处境** 类选题，爆款写法：痛点开头 → 分论点并列 → 案例佐证 → 金句升华 → 引导关注转发。

## 内容

```
wechat-viral-article/
└── SKILL.md          # Skill 主体（含写作骨架、选题、结构、案例库、降AI率技巧）
wechat-viral-article.skill   # 打包好的可安装文件
```

## 安装与使用

**方式一：安装打包文件**

下载 `wechat-viral-article.skill`，在 Claude Code / 桌面端导入即可。

**方式二：放入 skills 目录**

把 `wechat-viral-article/` 文件夹放进你的 Claude skills 目录，重启后生效。

安装后，当你说「帮我写一篇关于 XX 的公众号文章」且语境偏职场 / 成长 / AI 时，技能会自动触发。

## 特点

- **完整输出**：标题（含备选）、封面文案、题记、开头、分论点正文、结尾引导
- **案例库**：影视剧角色 / 名人经历 / 心理学概念 / 经典语录四类素材
- **降 AI 率技巧**：内置一套让文章更「像人写的」的写法建议
